JavaTM Advanced Imaging Image I/O Tools Binary Builds

This page contains pointers to binary builds of JavaTM Advanced Imaging Image I/O Tools. The following build types are available:

  • release: the final build for each JAI Image I/O Tools version;
  • stable: early access builds including alpha, beta, and rc milestones; and;  and
  • daily: builds which reflect the state of the CVS repository on the indicated date.

The builds are in reverse chronological order within each of these build categories.

For final release and milestone (alpha, beta, rc) builds there are four types of installation bundles for each platform:

  • bundles of the form *-jdk* for installation into a JDK;
  • bundles of the form *-jre* for installation into a JRE;
  • bundles of the form *jar.zip for auto-installation into a browser; and
  • bundles of the form *<platform>.tar.gz or *<platform>.exe for installation at a location specified by the CLASSPATH environment variable.

Stable and daily builds are provided as Zip archives only. To use these builds, the jar files contained in the archive must be placed on the CLASSPATH, the .so files on the LD_LIBRARY_PATH (Solaris and Linux), and the .dll files on the PATH (Windows). Javadoc bundles of the form *doc*.zip are provided for all bundles.
