This is the parent project for all Java Advanced Imaging Image I/O Tools related sub-projects on java.net. These sub-projects include: jai-imageio-core (the core Java Advanced Imaging Image I/O Tools API), and jai-imageio-demos (the Java Advanced Imaging Image I/O Tools demonstration programs). See the complete list of sub-projects at the bottom of this page.

Note that there is no source code in this parent project; source code files for Java Advanced Imaging Image I/O Tools related sub-projects are in the sub-projects themselves.

Feedback and Discussion

We have created four Java Advanced Imaging Image I/O Tools mailing lists in this project: interest 'at' jai-imageio.dev.java.net, announce 'at' jai-imageio.dev.java.net, issues 'at' jai-imageio.dev.java.net, and cvs 'at' jai-imageio.dev.java.net (replace 'at' with @). Click here to subscribe to any of these lists. The interest list is an open mailing list for discussing all aspects of Java Advanced Imaging Image I/O Tools, the Java Advanced Imaging Image I/O Tools API core, demo programs, etc. The announce list is a low-volume moderated mailing list for announcing Java Advanced Imaging Image I/O Tools related news and information. The issues list is for automated issues tracking (for issues in the parent jai-imageio web pages only), and the cvs list is for automated CVS change messages (for changes to the parent jai-imageio web pages only).

For those who prefer a forum to a discussion mailing list, we have created the Java Advanced Imaging Image I/O Tools forum, hosted as part of the JavaDesktop Community. Messages posted on the forum are cross-posted to interest 'at' jai-imageio.dev.java.net and vice-versa.

Finally, each sub-project has its own cvs and issues mailing list. The cvs lists are for automated CVS change messages and discussion of those changes. The issues lists are for automated issue tracking.
